# SistemaInventario
El presente proyecto consiste en el desarrollo de un Sistema de Gestión de Inventarios implementado en Python, utilizando el paradigma de Programación Orientada a Objetos (POO).

El sistema permite administrar productos dentro de una tienda mediante operaciones básicas de gestión como añadir, eliminar, actualizar, buscar y visualizar productos.

Este proyecto tiene como finalidad aplicar conceptos fundamentales de encapsulamiento, modularidad, manejo de estructuras de datos personalizadas y control de errores, así como reforzar el uso de herramientas de desarrollo como PyCharm y GitHub.
